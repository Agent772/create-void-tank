# Create Void Tank

Pipe it in and forget it. The Create Void Tank is a compact mod that adds a single block — the **Void Tank** — which accepts fluid from any side and instantly destroys it. It never fills up, never backs up your pipes, and scales into multi-block structures just like Create's own fluid tanks.

Quick Features
--------------
- New block: **Void Tank** — accepts fluids from any side and voids them instantly
- Multi-block connectivity (up to 3×3 × 16 tall), mirrors Create's fluid tank behaviour
- Toggle glass windows by right-clicking with a Wrench
- Three activation modes configurable per-server:
  - `ALWAYS_ACTIVE` — voids continuously (default)
  - `REQUIRES_HEAT` — needs a heat source below (campfire, fire, lava, or Blaze Burner)
  - `REQUIRES_REDSTONE` — only voids while receiving a redstone signal
- When heat mode is active, the minimum heat level is configurable (Smouldering / Kindled / Seething)


Crafting Recipe
---------------

```
C O C
O E O   →  Void Tank
C O C
```

- **C** — Copper Sheet (`create:copper_sheet`)
- **O** — Obsidian
- **E** — Ender Pearl
